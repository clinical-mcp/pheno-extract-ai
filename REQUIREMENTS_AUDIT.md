# Requirements audit

`requirements.txt` was generated by scanning all Python files in `extraction_scripts/` and `evaluation_scripts/` for non-standard-library imports.

Included packages:

- `anthropic` for Claude API scripts
- `openai` for OpenAI-compatible API scripts, including GPT/Gemini-compatible endpoints as written
- `xai-sdk` for Grok/xAI scripts (`import xai_sdk`)
- `httpx` for local/Ollama pipeline HTTP calls
- `pandas`, `numpy`, and `scipy` for evaluation/analysis scripts

Standard-library modules such as `csv`, `json`, `os`, `re`, `argparse`, and `collections` are not listed in `requirements.txt`.
